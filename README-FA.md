# LectureFlow 🎙

پلتفرم تبدیل گفتار به متن و تحلیل هوشمند محتوای آموزشی با استفاده از هوش مصنوعی

فایل صوتی کلاس، سخنرانی یا هر محتوای صوتی دیگر را آپلود کنید تا به‌صورت خودکار موارد زیر تولید شوند:

* متن کامل (Transcript)
* استخراج اصطلاحات تخصصی
* استخراج فرمول‌ها
* خلاصه‌سازی هوشمند توسط هوش مصنوعی

این پروژه برای دانشجویان، پژوهشگران، مدرسین و هر فردی که به تحلیل محتوای صوتی نیاز دارد طراحی شده است.

## راه‌اندازی

### ۱. دریافت کلیدهای API

#### Groq (تبدیل گفتار به متن)

1. وارد https://console.groq.com شوید.
2. حساب کاربری ایجاد کنید.
3. بخش API Keys را باز کنید.
4. یک کلید جدید بسازید.
5. کلید خود را کپی کنید (`gsk_...`).

#### Google Gemini (تحلیل محتوا)

1. وارد https://aistudio.google.com شوید.
2. وارد حساب گوگل خود شوید.
3. روی Get API Key کلیک کنید.
4. یک API Key جدید بسازید.
5. کلید خود را کپی کنید (`AIza...`).

#### NVIDIA NIM (استخراج فرمول)

1. وارد https://build.nvidia.com شوید.
2. حساب کاربری ایجاد کنید.
3. بخش API Keys را باز کنید.
4. یک کلید جدید ایجاد کنید.
5. کلید خود را کپی کنید (`nvapi-...`).

---

### ۲. استقرار روی Cloudflare Workers

1. وارد https://workers.cloudflare.com شوید.
2. یک Worker جدید ایجاد کنید.
3. کد پیش‌فرض را با فایل `worker.js` جایگزین کنید.
4. پروژه را Deploy کنید.

---

### ۳. تنظیم متغیرهای محرمانه

در پنل Worker وارد بخش زیر شوید:

Settings → Variables and Secrets

سپس متغیرهای زیر را ایجاد کنید:

| نام            | مقدار     |
| -------------- | --------- |
| GROQ_API_KEY   | gsk_...   |
| GEMINI_API_KEY | AIza...   |
| NVIDIA_API_KEY | nvapi-... |

پس از ذخیره، Worker را مجدداً Deploy کنید.

---

### ۴. استفاده از LectureFlow

آدرس Worker خود را باز کنید:

```text
https://your-worker.workers.dev
```

فایل صوتی را آپلود کنید و منتظر بمانید تا تحلیل به‌صورت خودکار انجام شود.
