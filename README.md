# LectureFlow
AI-powered lecture assistant — turns audio recordings into transcripts, formulas, terms & summaries.

Built with: Groq Whisper · NVIDIA NIM · Google Gemini · Cloudflare Workers

Setup:
1. Deploy worker.js to Cloudflare Workers
2. Add environment variables:
   - GROQ_API_KEY
   - GEMINI_API_KEY  
   - NVIDIA_API_KEY
3. Done!
